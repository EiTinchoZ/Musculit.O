# ROUTINE.md

Rutina fuente de verdad de Musculit.O. Esta es la version que se debe modelar en la app salvo que el usuario confirme un cambio.

## Reglas generales

- Descanso entre sets: `2 minutos`
- Intensidad: cerca del fallo, sin romper tecnica
- Rango dominante: `8-10 reps`
- Cardio post-entreno: `20 minutos de escaladora a baja intensidad` en todos los dias de entrenamiento
- Objetivo: recomposicion corporal con foco en ganar algo de masa, verte mas marcado y sostener consistencia real

## Estructura semanal

| Dia | Enfoque | Cardio | Observacion |
| --- | --- | --- | --- |
| Lunes | Piernas | 20 min | Dia pesado |
| Martes | Upper Body | 20 min | Dia mas largo de la semana |
| Miercoles | Descanso | No | Recuperacion total |
| Jueves | Piernas | 20 min | Se repite la base del lunes |
| Viernes | Push | 20 min | Compatible con ir al gym con la novia |
| Sabado | Pull | 20 min | Luego cardio con la novia |
| Domingo | Descanso | No | Recuperacion total |

## Lunes y Jueves - Piernas

La misma rutina aplica a ambos dias.

1. `Leg curl` - `3 x 8-10`
2. `Calf raises` - `3 x 10`
   Estas dos van en `biserie`: haces un set de leg curl y enseguida un set de calf raises antes de descansar.
3. `Romanian deadlift` - `2 x 8`
4. `Smith machine squats` - `3 x 8`
5. `Bulgarian split squat` - `2 x 8` por pierna
6. `Leg extension` - `2 x 8-10`
7. `Hip thrust` - `2-3 x 8`
8. `Abductor machine` - `1 set al fallo`
   Va al final de toda la sesion como finisher.

## Martes - Upper Body

### Shoulder

1. `Dumbbell shoulder press` - `2 x 8`
2. `Cable lateral raises` - `2 x 10` por brazo

### Chest

3. `Machine incline chest press` - `3 x 8`
4. `Machine chest flyes` - `2 x 8`

### Back

5. `Lat pulldown` - `2 x 8`
6. `Cable rows` - `2 x 8` por brazo
7. `Barbell cable rows` - `2 x 8`

### Rear delts

8. `Reverse machine flyes` - `1 x 10`

### Tricep

9. `Cable overhead tricep extension` - `2 x 8`
10. `Katana tricep extension` - `2 x 8`

### Bicep

11. `Dumbbell incline curl` - `2 x 8`
12. `Dumbbell hammer curl` - `2 x 8`

## Viernes - Push

### Shoulder

1. `Dumbbell shoulder press` - `2 x 8`
2. `Cable lateral raises` - `3 x 10` por brazo

### Chest

3. `Machine incline chest press` - `2 x 8`
4. `Dumbbell flat chest press` - `2 x 8`
5. `Machine chest flyes` - `2 x 8`

### Tricep

6. `Cable overhead tricep extension` - `2 x 8`
7. `Katana tricep extension` - `2 x 8`

## Sabado - Pull

### Back

1. `Lat pulldown` - `2 x 8`
2. `Cable rows` - `2 x 8` por brazo
3. `Barbell cable rows` - `2 x 8`
4. `Upright single arm rows` - `2 x 8`

### Rear delts

5. `Reverse machine flyes` - `2 x 10`

### Traps

6. `Dumbbell shrugs` - `2 x 8`

### Bicep

7. `Dumbbell incline curl` - `1 x 8`
8. `Preacher curl en maquina o mancuerna` - `2 x 8`
9. `Dumbbell hammer curl` - `2 x 8`

## Notas operativas

- `Upper Body` suele tomar entre `1h 15m` y `1h 30m`.
- `Push` debe mantenerse en torno a `50-60 minutos`.
- `Pull` debe mantenerse cerca de `1 hora`.
- En piernas, `Leg curl + Calf raises` se hace en biserie segun la rutina actual.
- Si un peso permite mas de `10 reps` limpias, hay margen para subirlo.
- La app debe permitir registrar pesos por `set` para ver aumento semanal y mensual real.
- La app debe incluir journal post-sesion para anotar sensaciones, tecnica, energia y observaciones.
